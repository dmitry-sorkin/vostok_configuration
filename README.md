# Конфигурация 3D принтера VOSTOK

Конфигурация 3D принтера VOSTOK для [экспериментальной версии Klipper'а](https://github.com/dmbutyugin/klipper/tree/generic-cartesian), поддерживающей прямое управление правой печатающей головой. По сравнению с классической конфигурацией, позволяет осуществлять сложные движения, при которых одновременно двигаются две печатающие головы. Это чуть-чуть ускоряет некоторые макросы типа парковки осей, а также позволяет очень быстро менять активную печатающую голову.

## Состояние

На текущий момент эта конфигурация очень слабо протестирована и требует доработки. Использовать её на основном рабочем принтере не стоит.

## Инструкция

Инструкция по применению конфигурации находится в файле printer.cfg

!!! Обратите внимание, что эту конфигурацию можно установить только на [экспериментальную версию Klipper'а](https://github.com/dmbutyugin/klipper/tree/generic-cartesian) !!!

## Как добавить свой electronics_*.cfg

Для этого надо сделать pull request, добавляющий этот файл в папку `user_electronics`. Обратите внимание, что файл не должен содержать конфигурации, кроме как именно под электронику
